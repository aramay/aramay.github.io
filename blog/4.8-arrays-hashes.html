4.8 Arrays and Hashes

Arrays

arrays are ordered, integer-indexed collections of any object. Each element in an array is associated with and referred to by an index. Ruby arrays can hold objects such as String, Integer, Fixnum, Hash, Symbol, even other Array objects.

Hashes

A Hash is a dictionary-like collection of unique keys and their values. Also called associative arrays, they are similar to Arrays, but where an Array uses integers as its index, a Hash allows you to use any object type.
