5.9 Enumerables

Enumerable#map

The Enumerable module provides a set of methods to traverse, search, sort and manipulate collections.

Without enumerables, we would have access index of an array to manipulate data.

With map, it Invokes the given block once for each element with a new array containing the values returned by the block.

Bang Methods -Destructive Methods

Methods that end in ! indicate that the method will modify the object it's called on. Ruby calls these "dangerous methods" because they change state.

We can extend this functionality by chaining other enumerables e.g. with_index
