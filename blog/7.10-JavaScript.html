Ruby Hashes vs. JavaScript Objects

JavaScript Objects

So Ruby and Javascript are fairly similar as they both use Object Oriented Programming.They can create these elements called Objects that represent some form of data as opposed to actions. This makes it much easier for the programmer to manipulate an objects data.

This post in simply an overview of the differences/similarities between Ruby Hashes and JavaScript Objects.

Ruby - Hashes

A Hash is a collection of key-value pairs like this: "employee" => "salary". It is similar to an Array, except that indexing is done via arbitrary keys of any object type, not an integer index.

We can create a Hash in Ruby using a literal { }.
